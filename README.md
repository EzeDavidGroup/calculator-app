# 🧮 Calculator App

## Overview

A modern glassmorphism calculator built with HTML, CSS and JavaScript.

The project has evolved through multiple versions with continuous improvements, including calculation history, Local Storage support, keyboard compatibility and a cleaner software architecture.

This project demonstrates responsive design, DOM manipulation, event handling and modern JavaScript development practices.

## Live Demo

[https://ezedavidgroup.github.io/calculator/](https://ezedavidgroup.github.io/calculator-app/)

## Features

- Basic arithmetic operations
- Responsive Glassmorphism UI
- Persistent calculation history
- Local Storage support
- History drawer
- Clear history
- Keyboard support
- Input validation
- Responsive mobile design

## Built With

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage API
